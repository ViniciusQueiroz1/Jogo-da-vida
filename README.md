# Jogo-da-vida
Jogo da Vida (Conway's Game of Life)
Implementação do clássico Jogo da Vida de Conway (1970) em C++, utilizando a arquitetura MVC (Model-View-Controller) para organizar melhor o código.
O projeto simula populações de células em uma grade, que evoluem ao longo das gerações seguindo regras simples que reproduzem o ciclo da vida.

---

Objetivo
Reproduzir as regras do autômato celular criado por John Conway.
Permitir que o usuário configure o tabuleiro inicial e acompanhe a evolução das gerações.
Explorar conceitos de estruturas de dados, matrizes, listas dinâmicas e boas práticas de programação.

---

Funcionalidades
Definição do tamanho da matriz (10x10 até 50x50).
Inserção e exclusão manual de células.
Opção de visualizar ou ocultar células mortas.
Tutorial interativo explicando as regras e funcionamento.
Configuração de velocidade da simulação.
Execução passo a passo ou apenas resultado final.
Salvar e carregar gerações.
Limpar ou excluir gerações salvas.
Encerramento seguro do programa.

---

Tecnologias Utilizadas
Linguagem: C++
Paradigma: Programação Estruturada + Modular (MVC)
Estruturas de Dados: Matrizes, Listas Dinâmicas
Entrada/Saída: Console (modo texto)

---

Instalação e Execução
Pré-requisitos
Compilador g++ (Linux, macOS ou Windows via MinGW/WSL).
Git instalado (opcional, para clonar o repositório).

Passo a Passo
Clone este repositório:
git clone [https://github.com/ViniciusQueiroz1/jogo-da-vida-cpp.git](https://github.com/ViniciusQueiroz1/Jogo-da-vida/tree/main/Jogo%20da%20vida)
cd jogo-da-vida-cpp

Compile todos os arquivos
g++ *.cpp -o JVIDA2-EJPV-Projeto    

Execute
./JVIDA2-EJPV-Projeto
